Installation & Other Documentation
version 1.0 by Alex Mangini Kolakube

Congratulations on your new Kolakube skin! Once you have downloaded the ZIP file from the Members Area, find it in the place on your computer where you saved it to and unzip it. After it has been unzipped, you should see a folder called nexis_10.

Follow the instructions below to install your new skin into Thesis.

Installation Instructions Install this skin in 4 simple steps!

Connect to your site via FTP and go to the following folder: /wp-content/themes/thesis_18/. Within the thesis_18 folder is the custom folder. Once you have found this folder, proceed to step 2.

Note: If you haven't renamed this folder from a fresh installation of Thesis, it will be called custom-sample.

Once you have located the thesis_18 folder, right click it and delete the entire custom (or custom-sample) folder. Once it is deleted, proceed to step 3.

Now, go to the folder on your computer you just downloaded titled nexis_10. Open it, and inside will be a folder called custom. Upload that folder into the thesis_18 directory (the place you just deleted the default custom folder from).

Still in your FTP program, open up the custom folder you just downloaded and CHMOD (change the file permissions) layout.css to 666.

Once you have done these things, you are done working in your FTP program. You can now proceed to step 4.

Login to your WordPress Admin Panel and navigate to the sidebar panel titled Thesis. Open the Thesis Options if they aren't open already and click Manage Options.

In the Upload Options column, go to the last upload box titled All Options. Click Browse and locate the file thesis-all-options-nexis.dat in the nexis_10 folder you received in your download. Upload it and save.

Nexis has been successfully installed! If you have any problems installing this skin, or these instructions are not clear enough, please visit our awesome support forums for one-on-one help.

Adjusting Column Widths How to safely adjust the width of your site.

Whenever you change the width of the content area in the Thesis Design Options, you will also need to make a little tweak in your CSS file. It's a pain, and we'll try to make this automatic in future releases.

So when you do change the width of the content column, it looks as if the sidebar breaks. As shown in the example to the left, the widget stretches past the gray sidebar area. Not good.

Luckily, it's an easy little CSS tweak. Open up custom.css and find: .custom #content_box

Within that element is the background image line. If you have any knowledge of faux columns, you just need to make a tweak of the background position on the background image.

background: url(images/sidebar.png) repeat-y 61.4em 0

The higher the number you make your content, the higher the bolded number should be.

The smaller the number you make your content, the smaller the bolded number should be.
